# Nuclear AutoClicker

Fast and easy to use Auto Clicker with a simple GUI.

## Features
- **Click Modes**: Spam or hold clicks continuously  
- **Mouse Buttons**: Supports left, right, and middle click  
- **Keyboard Support**: Can spam any key on the keyboard  
- **Custom Hotkeys**: Set your own start/stop hotkey for quick control  
- **Lightweight & Fast**: Minimal resource usage with responsive performance  

## Note on Antivirus Warnings
Because Nuclear AutoClicker automates mouse and keyboard input, some antivirus programs (including Windows Defender) may flag it as potentially unsafe.  
This is a **false positive**,  the application does not contain malware and is safe to use.  
If you encounter a warning, you can safely allow the program to run.
