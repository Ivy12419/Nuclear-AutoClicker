# Nuclear Auto Clicker

Fast and easy to use Auto Clicker with a simple GUI.

## Features
- **Click Modes**: Spam or hold clicks continuously  
- **Mouse Buttons**: Supports left, right, and middle click  
- **Keyboard Support**: Can spam any key on the keyboard  
- **Custom Hotkeys**: Set your own start/stop hotkey for quick control  
- **Lightweight & Fast**: Minimal resource usage with responsive performance
